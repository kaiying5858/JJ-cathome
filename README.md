# 猫窝之家 - CatNest Home

这是一个展示和销售高品质猫窝的网站。网站采用简洁现代的设计风格，为猫咪主人提供舒适、美观的猫窝选择。

## 项目结构
- `index.html`: 主页面
- `css/`: 样式文件目录
  - `style.css`: 主样式文件
- `js/`: JavaScript文件目录
  - `main.js`: 主脚本文件
- `images/`: 图片资源目录
- `assets/`: SVG图标等资源目录

## 功能特点
- 响应式设计，适配各种设备
- 现代简约的用户界面
- 产品展示和详情页面
- 简单的购物车功能

## 开发技术
- HTML5
- CSS3
- JavaScript
